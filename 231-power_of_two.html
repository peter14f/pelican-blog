231-power_of_two

Feb 11 2016 Tags Power

LeetCode Problem Link

Pretty straightforward problem. Just remember that a power of 2 number cannot be negative.

public boolean isPowerOfTwo(int n) {
    // number of bits set
    int numBits = 0;

    do {
        if ((n & 1) == 1) {
            numBits++;
            if (numBits > 1)
                return false;
        }
        n = n >> 1;
    } while (n != 0);

    return (numBits==1);
}
