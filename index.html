069-sqrt

Nov 02 2015

LeetCode Problem Link

My solution a O(n) time. Basically I am looking for a number starting from 1 whose squared value is equal to exceed the input value x. If the squared value exceeds x, then we know that i - 1 is the sqrt integer value.

Overflow may occur so product is a long instead of an int.

public int mySqrt(int x) {

    if (x < 0)
        throw new IllegalArgumentException();

    for (int i=1; i<=x; i++) {
        long product = (long)i * i;

        if (product == x) {
            return i;
        }
        else if (product > x) {
            return i - 1;
        }
    }

    return 0;
}
