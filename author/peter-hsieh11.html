237-delete_node_in_a_linked_list

Feb 15 2016 Tags Singly Linked Lists

LeetCode Problem Link

The problem requires that you think outside of the box. You only have access to the nodes starting at the node you want to delete. So all we can do is copy the values from the following nodes.

public void deleteNode(ListNode node) {

    if (node == null)
        return;

    ListNode cur = node;
    ListNode prev = null;

    while (cur != null) {
        if (cur.next == null && prev != null) {
            prev.next = null;
        }
        else {
            cur.val = cur.next.val;
        }

        prev = cur;
        cur = cur.next;
    }
}

It's impossible to delete if the node given is the tail node.

234-palindrome_linked_list

Feb 12 2016 Tags Palindromes / Linked Lists

LeetCode Problem Link

The first solution using a stack is the most straightforward one. It takes O(n) time and O(n) space.

public boolean isPalindrome(ListNode head) {
    if (head == null)
        return true;

    Stack<ListNode> stk = new Stack<ListNode>();

    ListNode tail = head;

    while (tail != null) {
        stk.push(tail);
        tail = tail.next;
    }

    ListNode cur = head;

    while (!stk.isEmpty()) {
        ListNode node = stk.pop();
        if (node.val != cur.val)
            return false;
        cur = cur.next;
    }

    return true;
}

233-number_of_digit_one

Feb 11 2016 Tags Math

LeetCode Problem Link

This problem is asking how many 1s are seen in non-negative integer <= n.

Here is the simplest solution that will get you TLE for a large n.

private int countOnes(int n) {
    int num = 0;

    while (n > 0) {
        if ((n % 10) == 1)
            num++;
        n = n / 10;
    }

    return num;
}

public int countDigitOne(int n) {
    int sum = 0;

    while (n > 0) {
        int num = countOnes(n);
        sum += num;
        n--;
    }

    return sum;
}

231-power_of_two

Feb 11 2016 Tags Power

LeetCode Problem Link

Pretty straightforward problem. Just remember that a power of 2 number cannot be negative.

public boolean isPowerOfTwo(int n) {
    // number of bits set
    int numBits = 0;

    do {
        if ((n & 1) == 1) {
            numBits++;
            if (numBits > 1)
                return false;
        }
        n = n >> 1;
    } while (n != 0);

    return (numBits==1);
}
