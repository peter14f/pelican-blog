LeetCode Problem Link

public int hammingDistance(int x, int y) {
    int z = x ^ y;
    int hammingDistance = 0;
    for (int i=0; i<32; i++) {
        if ((z & 1) == 1) {
            hammingDistance++;
        }
        z = z >>> 1;
    }
    return hammingDistance;
}

477-total_hamming_distance

Jul 13 2018 Tags Couting Bits / Bit Manipulation

LeetCode Problem Link

Imagine each number is just zero or one, the total hamming distance is just (number of ones)*(number of zeros)

Do that for each bit and sum up the result at each bit.

public int totalHammingDistance(int[] nums) {
    int totalDistance = 0;

    for (int i=0; i<32; i++) {
        int numOnes = 0;
        int numZeros = 0;
        for (int j=0; j<nums.length; j++) {
            if ((nums[j] & 1) == 1) {
                numOnes++;
            } else {
                numZeros++;
            }
            nums[j] = nums[j] >>> 1;
        }
        totalDistance += (numOnes * numZeros);
    }

    return totalDistance;
}

868-transpose_matrix

Jul 10 2018 Tags 2D Matrix

LeetCode Problem Link

We cannot do this in place because the input matrix could be rectangular instead of square.

public int[][] transpose(int[][] A) {
    int m = A.length;

    if (m < 1) {
        return new int[0][0];
    }

    int n = A[0].length;

    int[][] B = new int[n][m];

    for (int i=0; i<m; i++) {
        for (int j=0; j<n; j++) {
            B[j][i] = A[i][j];
        }
    }

    return B;
}

647-palindromic_substrings

Jul 10 2018 Tags Palindromes / Dyanamic Programming

LeetCode Problem Link

First thought is to go through all substrings starting and ending at each possible locations - this takes O(n^2) time. And then for each substring check if it's palindromic.

public int countSubstrings(String s) {

    char[] ch = s.toCharArray();
    int total = 0;

    // n^2 to go through all substrings
    for (int start=0; start<ch.length; start++) {
        for (int end=start; end<ch.length; end++) {
            if (isPalindrome(ch, start, end)) {
                total++;
            }
        }
    }

    return total;
}

private boolean isPalindrome(char[] ch, int start, int end) {
    while (start <= end) {
        if (ch[start] != ch[end]) {
            return false;
        }
        start++;
        end--;
    }
    return true;
}
