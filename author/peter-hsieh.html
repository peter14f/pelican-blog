868-transpose_matrix

Mar 21 2018 Tags 2D Matrix

LeetCode Problem Link

We cannot do this in place because the input matrix could be rectangular instead of square.

public int[][] transpose(int[][] A) {
    int m = A.length;

    if (m < 1) {
        return new int[0][0];
    }

    int n = A[0].length;

    int[][] B = new int[n][m];

    for (int i=0; i<m; i++) {
        for (int j=0; j<n; j++) {
            B[j][i] = A[i][j];
        }
    }

    return B;
}

739-daily_temperatures

Mar 20 2018 Tags Stacks

LeetCode Problem Link

The trivial solution would take O(n2) time. This problem reminds me of 84-largest_rectangle.

public int[] dailyTemperatures2(int[] temperatures) {
    Stack<Integer> stk = new Stack<Integer>(); // stores the index
    int[] ans = new int[temperatures.length];
    int i=0;
    while (!stk.isEmpty() || i<temperatures.length) {
        if (stk.isEmpty()) {
            stk.push(i);
            i++;
            continue;
        }

        if (i >= temperatures.length || temperatures[i] > temperatures[stk.peek()]) {
            int index = stk.pop();
            if (i >= temperatures.length) {
                ans[index] = 0;
            } else {
                ans[index] = i-index;
            }
        } else {
            stk.push(i);
            i++;
        }
    }
    return ans;
}

714-best_time_to_buy_and_sell_stock_with_transaction_fee

Mar 20 2018 Tags Best Time To Buy/Sell / Dynamic Programming

LeetCode Problem Link

// 1  3  2  8  4  9

// 0  0  0  5  5  8
//-1 -1 -1 -1  1  1
public int maxProfit(int[] prices, int fee) {
    int n = prices.length;

    int balanceWithStock = -prices[0];
    int balanceWithNoStock = 0;

    for (int i=1; i<n; i++) {
        balanceWithNoStock = Math.max(balanceWithNoStock, balanceWithStock + prices[i] - fee);
        balanceWithStock = Math.max(balanceWithStock, balanceWithNoStock - prices[i]);
    }

    return balanceWithNoStock;
}
