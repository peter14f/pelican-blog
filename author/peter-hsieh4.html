069-sqrt

Nov 02 2015

LeetCode Problem Link

My solution a O(n) time. Basically I am looking for a number starting from 1 whose squared value is equal to exceed the input value x. If the squared value exceeds x, then we know that i - 1 is the sqrt integer value.

Overflow may occur so product is a long instead of an int.

public int mySqrt(int x) {

    if (x < 0)
        throw new IllegalArgumentException();

    for (int i=1; i<=x; i++) {
        long product = (long)i * i;

        if (product == x) {
            return i;
        }
        else if (product > x) {
            return i - 1;
        }
    }

    return 0;
}

066-plus_one

Oct 29 2015

LeetCode Problem Link

The problem is simple when the input array does not just contain only the number 9.

When you do, you have to allocate an array of size n+1 and set the first element to 1.

public int[] plusOne(int[] digits) {
    int n = digits.length;

    if (digits[n-1] != 9) {
        digits[n-1] +=1;
        return digits;
    }
    else {
        digits[n-1] = 0;

        int i = n-2;
        while (i >= 0) {
            if (digits[i] != 9) {
                digits[i] += 1;
                return digits;
            }
            i--;
        }

        int[] newDigits = new int[n+1];

        newDigits[0] = 1;
        return newDigits;
    }
}
