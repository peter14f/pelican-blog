242-valid_anagram

Feb 17 2016 Tags Anagrams

LeetCode Problem Link

Compare the sorted versions of s and t would do.

public boolean isAnagram(String s, String t) {

    if (s.length() != t.length())
        return false;

    char[] sArr = s.toCharArray();
    Arrays.sort(sArr);
    char[] tArr = t.toCharArray();
    Arrays.sort(tArr);

    for (int i=0; i<sArr.length; i++) {
        if (sArr[i] != tArr[i])
            return false;
    }

    return true;
}

239-sliding_window_maximum

Feb 16 2016 Tags Sliding Windows / Heap

LeetCode Problem Link

The most straightforward solution take O(nk) time which simply uses two nested for loops to try to look for the max element in the current sliding window.

// can assume k to be valid
public int[] maxSlidingWindow(int[] nums, int k) {
    if (nums.length == 0)
        return nums;

    int[] output = new int[nums.length-k+1];

    for (int i=0; i+k-1 < nums.length; i++) {
        int max = nums[i];
        for (int j=1; j<k; j++) {
            if (nums[i+j] > max)
                max = nums[i+j];
        }
        output[i] = max;
    }

    return output;
}

public int[] productExceptSelf(int[] nums) {
    if (nums.length == 0)
        return nums;

    int[] output = new int[nums.length];

    for (int i=0; i<output.length; i++) {
        int p = 1;
        for (int j=0; j<nums.length; j++) {
            if (i==j)
                continue;
            p = p*nums[j];
        }
        output[i] = p;
    }

    return output;
