001 Two Sum

Aug 25 2015 Tags Hash Table / X-Sum

LeetCode Problem Link

There are many ways the problem can be asked in an interview. Some will simply ask you to return the two numbers. LeetCode instead wants to know the location of two numbers found in the input array. An int array of size two needs to be returned. And index + 1 is stored in the returned array where index is the index of the input array.

An example:

nums = {0, 4, 3, 0}
target = 0

{1, 4} should be returned since adding the first number 0 and the fourth number 0 gives you 0 (target).
