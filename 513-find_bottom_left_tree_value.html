513-find_bottom_left_tree_value

Mar 18 2018 Tags Binary Trees / Binary Tree Traversal

LeetCode Problem Link

First thing that came to my mind is that level-order traversal could help. I need two queues q and q2.

At each iteration of the while I traverse all nodes in q and store each node's children (left first, then right) into q2.

At the end of each iteration of the while loop, if q2 is empty, then I know q contains the nodes in the very last level. And the very first element would give me the left most node.

public int findBottomLeftValue(TreeNode root) {

    ArrayList<TreeNode> q = new ArrayList<TreeNode>();
    q.add(root);
    ArrayList<TreeNode> q2 = new ArrayList<TreeNode>();

    while (!q.isEmpty()) {
        for (int i=0; i<q.size(); i++) {
            TreeNode n = q.get(i);
            if (n.left != null) {
                q2.add(n.left);
            }
            if (n.right != null) {
                q2.add(n.right);
            }
        } // traverse q

        if (q2.isEmpty()) {
            // q contains nodes in the last level
            break;
        } else {
            q = q2;
            q2 = new ArrayList<TreeNode>();
        }
    }

    return q.get(0).val;
}
