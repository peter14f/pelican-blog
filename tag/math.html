238-product_of_array_except_self

Feb 16 2016 Tags Math

LeetCode Problem Link

Here is the straightforward O(n2) time solution. Well, the problem statement already asks us to do it in O(n) time, so this obviously will get us TLE.

public int[] productExceptSelf(int[] nums) {
    if (nums.length == 0)
        return nums;

    int[] output = new int[nums.length];

    for (int i=0; i<output.length; i++) {
        int p = 1;
        for (int j=0; j<nums.length; j++) {
            if (i==j)
                continue;
            p = p*nums[j];
        }
        output[i] = p;
    }

    return output;
}

233-number_of_digit_one

Feb 11 2016 Tags Math

LeetCode Problem Link

This problem is asking how many 1s are seen in non-negative integer <= n.

Here is the simplest solution that will get you TLE for a large n.

private int countOnes(int n) {
    int num = 0;

    while (n > 0) {
        if ((n % 10) == 1)
            num++;
        n = n / 10;
    }

    return num;
}

public int countDigitOne(int n) {
    int sum = 0;

    while (n > 0) {
        int num = countOnes(n);
        sum += num;
        n--;
    }

    return sum;
}

050-pow_x_n

Oct 11 2015 Tags Power / Math

LeetCode Problem Link

Use recursion to avoid re-computation.

x4 = x2 * x2
x5 = x4 * x
public double myPow(double x, int n) {

    if (n==0)
        return 1;
    else if (n==1)
        return x;
    else if (n==-1)
        return 1/x;

    boolean negative = false;

    if (n<0)
        negative = true;

    if (n%2==0) {
        // even power
        int halfPower = n/2;
        double halfPowerAns = myPow(x, halfPower);
        return halfPowerAns*halfPowerAns;
    }
    else {
        // odd power
        if (negative) {
            int evenPower = n + 1;

            return myPow(x, evenPower) * (1/x);
        }
        else {
            int evenPower = n - 1;
            return myPow(x, evenPower) * x;
        }
    }
}
