374-guess_number_higher_or_lower

Mar 17 2018 Tags Binary Search

LeetCode Problem Link

/* The guess API is defined in the parent class GuessGame.
   @param num, your guess
   @return -1 if my number is lower, 1 if my number is higher, otherwise return 0
   int guess(int num); */

public class Solution extends GuessGame {
    public int guessNumber(int n) {
        int l = 1;
        int h = n;

        while (l <= h) {
            int m = l + (h-l)/2;
            int result = guess(m);
            if (result < 0) {
                // m is too high
                h = m - 1;
            } else if (result > 0) {
                // m is too low
                l = m + 1;
            } else {
                return m;
            }
        }

        return 0;
    }
}

278-first_bad_version

Feb 20 2016 Tags Binary Search

LeetCode Problem Link

There are n versions labeled from 1 to n.

O(n) solution would be to try call isBadVersion() for 0, 1, 2, ... .

The O(logn) solution uses binary search.

public int firstBadVersion(int n) {
    int l = 1;
    int h = n;

    int minFound = n;

    while (l <= h) {
        int m = l + (h-l)/2;

        boolean bad = isBadVersion(m);

        if (bad) {
            minFound = Math.min(minFound, m);
            // could have a lower bad version
            h = m - 1;
        }
        else {
            // could have a higher bad version
            l = m + 1;
        }
    }

    return minFound;
}

035-search_insert_position

Sep 27 2015 Tags Binary Search

LeetCode Problem Link

Use binary search. If the number is not found, the l pointer actually will be the correct index if the number were to be inserted.

public int searchInsert(int[] nums, int target) {

    int l = 0;
    int h = nums.length - 1;

    while (l <= h) {

        int middle = l + (h-l)/2;
        int middleNum = nums[middle];

        if (middleNum == target) {
            return middle;
        }
        else if (middleNum > target) {
            // look in the left part
            h = middle - 1;
        }
        else {
            // look in the right part
            l = middle + 1;
        }

    }

    // not found, and now l == h + 1
    return l;
}
