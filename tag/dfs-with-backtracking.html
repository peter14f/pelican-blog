The password has n digits. Each digit has k different choices. The we know the total possible number passwords is k^n.

We are asked to generate a shortest possible string that covers all the possible passwords given k and n.

public String crackSafe(int n, int k) {
    StringBuilder sb = new StringBuilder();
    int total = (int)Math.pow(k, n);
    // password has n digits
    for (int i=0; i<n; i++) {
        sb.append('0');
    }

    Set<String> visited = new HashSet<>();
    visited.add(sb.toString());

    dfs(sb, total, visited, n, k);
    return sb.toString();
}

For most DFS with backtracking problems, you maintain one path. This problem requires maintaining k paths.

public boolean canPartitionKSubsets(int[] nums, int k) {
    if (nums.length < k) {
        return false;
    }

    int total = 0;
    for (int num : nums) {
        total += num;
    }

    if (total % k != 0) {
        return false;
    }

    Arrays.sort(nums);
    int target = total/k;

    return dfsCanPartitionKSubsets(nums, nums.length - 1, new int[k], target);
}
