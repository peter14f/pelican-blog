152-maximum_product_subarray

Jan 03 2016 Tags Subarray / Memoization

LeetCode Problem Link

The straightforward solution takes O(n 2) time, but will get you TLE on large test case.

public int maxProduct(int[] nums) {

    int n = nums.length;
    int max = nums[0];

    for (int i=0; i<n; i++) {
        int product = nums[i];
        max = Math.max(max, product);

        for (int j=i+1; j<n; j++) {
            product = product * nums[j];
            max = Math.max(max, product);
        }
    }

    return max;
}
