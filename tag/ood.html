a02-design_a_deck_of_cards

Mar 03 2016 Tags OOD

Typical design would need a Deck class and a Card class.

What methods does the Deck class need?

int numCardsLeft() - return how many cards are left to be dealt

void shuffle(int n) - swap the locations of n pairs of card

Card deal() - returns the next card
