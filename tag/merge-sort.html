public int countRangeSum(int[] nums, int lower, int upper) {
    int n = nums.length;

    int count = 0;

    for (int i=0; i<n; i++) {
        int curSum = nums[i];

        if (curSum >= lower && curSum <= upper)
            count++;

        for (int j=i+1; j<n; j++) {
            curSum += nums[j];

            if (curSum >= lower && curSum <= upper)
                count++;
        }
    }

    return count;
}

315-count_of_smaller_numbers_after_self

Feb 26 2016 Tags Merge Sort / Divide And Conquer

LeetCode Problem Link

The straightforward solution take O(n :superscript:2) time.

public List<Integer> countSmaller(int[] nums) {
    List<Integer> ans = new ArrayList<Integer>();

    int n = nums.length;

    for (int i=0; i<n; i++) {
        int cnt = 0;
        for (int j=i+1; j<n; j++) {
            if (nums[j] < nums[i])
                cnt++;
        }
        ans.add(cnt);
    }

    return ans;
}
