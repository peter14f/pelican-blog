Jan 17 2016 Tags sql / left join / right join

LeetCode Problem Link

Table - Person PersonId FirstName LastName PersonId is the primary key column for this table.

Table - Address AddressId PersonId City State AddressId is the primary key column for this table.

Write a SQL query for a report that provides the following information for each person in the Person table, regardless if there is an address for each of those people:

FirstName, LastName, City, State

select FirstName, LastName, City, State from
Person left join Address on
Person.PersonId = Address.PersonId;
