493-reverse_pairs

Mar 17 2018 Tags Merge Sort / Divide And Conquer / BIT / Self-Balancing BST

LeetCode Problem Link

This problem asks us to find the number of distinct (i, j) pairs such that - i < j - nums[i] > 2*nums[j]

The straightforward solution takes O(n^2) time.

public int reversePairs(int[] nums) {
    int n = nums.length;
    int ans = 0;
    for (int i=0; i<n; i++) {
        for (int j=i+1; j<n; j++) {
            long numI = nums[i];
            long numJ = nums[j];
            if (numI > 2*numJ) {
                ans++;
            }
        }
    }
    return ans;
}

327-count_of_range_sum

public int countRangeSum(int[] nums, int lower, int upper) {
    int n = nums.length;

    int count = 0;

    for (int i=0; i<n; i++) {
        int curSum = nums[i];

        if (curSum >= lower && curSum <= upper)
            count++;

        for (int j=i+1; j<n; j++) {
            curSum += nums[j];

            if (curSum >= lower && curSum <= upper)
                count++;
        }
    }

    return count;
}

315-count_of_smaller_numbers_after_self

Feb 26 2016 Tags Merge Sort / Divide And Conquer / BIT / Self-Balancing BST / Segment Tree

LeetCode Problem Link

The straightforward solution take O(n :superscript:2) time.

public List<Integer> countSmaller(int[] nums) {
    List<Integer> ans = new ArrayList<Integer>();

    int n = nums.length;

    for (int i=0; i<n; i++) {
        int cnt = 0;
        for (int j=i+1; j<n; j++) {
            if (nums[j] < nums[i])
                cnt++;
        }
        ans.add(cnt);
    }

    return ans;
}

You guessed it! This will get you TLE on large test case.
