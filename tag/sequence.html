038-count_and_say

Sep 27 2015 Tags Sequence / Recursion / Dynamic Programming

LeetCode Problem Link

The first number in the sequence is 1. The following numbers in the sequence simply tries to count the previous number in the sequence. I am showing the recursive approach here. The DP approach is practically the same, just doing it bottom-up.

public String countAndSay(int n) {

    if (n == 1) {
        return "1";
    }
    else {
        String prevNum = countAndSay(n-1);
        char[] arr = prevNum.toCharArray();
        char prev = arr[0];
        int cnt = 1;
        StringBuffer sb = new StringBuffer();

        for (int i=1; i<arr.length; i++) {

            if (arr[i] == prev) {
                cnt++;
            }
            else {
                sb.append(cnt);
                sb.append(prev);
