173-binary_search_tree_iterator

Jan 17 2016 Tags Binary Search Trees

LeetCode Problem Link

We asked to implement the inorder traversal iterator of a binary tree. The only field the Iterator needs is a Stack.

public class BSTIterator {
    Stack<TreeNode> stk;

    public BSTIterator(TreeNode root) {
        stk = new Stack<TreeNode>();
        while (root != null) {
            stk.push(root);
            root = root.left;
        }
    }

    /** @return whether we have a next smallest number */
    public boolean hasNext() {
        return !stk.isEmpty();
    }

    /** @return the next smallest number */
    public int next() {
        TreeNode node = stk.pop();
        TreeNode t = node.right;

        while (t != null) {
            stk.push(t);
            t = t.left;
        }

        return node.val;
    }
}

108-convert_sorted_array_to_binary_search_tree

Nov 26 2015 Tags Binary Search Trees / Recursion

LeetCode Problem Link

Use recursion. The middle element is already the root. This is because we're trying to build a height balanced BST. Otherwise, we could have simply build a linked list.

public TreeNode sortedArrayToBST(int[] nums) {

    if (nums.length == 0)
        return null;

    return sortedArrayToBST(nums, 0, nums.length - 1);

}

private TreeNode sortedArrayToBST(int[] nums, int low, int high) {

    int middle = low + (high-low)/2;

    TreeNode root = new TreeNode(nums[middle]);

    if (middle - 1 >= low) {
        root.left = sortedArrayToBST(nums, low, middle - 1);
    }

    if (middle + 1 <= high) {
        root.right = sortedArrayToBST(nums, middle + 1, high);
    }

    return root;
}

public boolean isValidBST(TreeNode root) {

    if (root == null)
        return true;

    return isValidBST(root.left, null, root) &&
            isValidBST(root.right, root, null);
