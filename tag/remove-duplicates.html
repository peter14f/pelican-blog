083-remove_duplicates_from_sorted_list

Nov 11 2015 Tags Remove Duplicates / Sorted List

LeetCode Problem Link

We're asked to keep only one node for each node in the sorted list that have duplicates.

Problem is not hard. When working wiht linked lists, always make sure that list is properly terminated.

public ListNode deleteDuplicates(ListNode head) {
    if (head==null) {
        return null;
    }

    ListNode cur = head;

    while (cur != null) {
        ListNode nextNode = cur.next;

        while (nextNode != null && nextNode.val == cur.val) {
            nextNode = nextNode.next;
        }

        cur.next = nextNode;
        cur = nextNode;
    }

    return head;
}

080-remove_duplicates_from_sorted_array_ii

Nov 10 2015 Tags Remove Duplicates / Sorted Array

LeetCode Problem Link

This is a followup problem for 026-remove_duplicates_from_sorted_array. The idea is the same, we just have to keep a counter to keep track of the number of times a number has show up in the sorted array consecutively.

public int removeDuplicates(int[] nums) {
    if (nums==null)
        return 0;
    if (nums.length < 3) {
        return nums.length;
    }

    int lastNum = nums[0];
    int cnt = 1;
    int index = 1;

    for (int i=1; i<nums.length; i++) {
        if (nums[i] == lastNum) {
            cnt++;
        }
        else {
            cnt = 1;
        }

        lastNum = nums[i];

        if (cnt < 3) {
            nums[index] = lastNum;
            index++;
        }
    }

    return index;
}

026-remove_duplicates_from_sorted_array

Sep 07 2015 Tags Remove Duplicates / Sorted Array

LeetCode Problem Link

The solution I came up with is to simply overwrite the array as we go.

i should start at 1 as nums[0] must be the smallest number.

public int removeDuplicates(int[] nums) {

    if (nums==null || nums.length==0)
        return 0;

    int i = 0;

    for (int j=1; j<nums.length; j++) {
        if (nums[j] != nums[i]) {
            i++;
            nums[i] = nums[j];
        }
    }

    return i+1;
}
