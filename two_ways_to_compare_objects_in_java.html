two_ways_to_compare_objects_in_java

Feb 28 2016

Say you have written a Java class like the following

class Edge {
    int val;
    boolean isLeft; // true if this is a left edge, false if it's a right edge
}

And you want to use a PriorityQueue to sort it such that the edge with a smaller value is considered smaller and if two edges are of the same value, the right edge should be considered smaller.

One way to make your Edge class implement the Comparable interface

class Edge implements Comparable<Edge> {
    int val;
    boolean isLeft;

    @Override
    public int compareTo(Edge o) {
        if (this.val < o.val) {
            return -1;
        } else if (this.val > o.val) {
            return 1;
        } else {
            if (!this.isLeft && o.isLeft) {
                return -1;
            } else if (this.isLeft && !o.isLeft) {
                return 1;
            } else {
                return 0;
            }
        }
    }
}

The other way would be to pass in a Comparator object when you are constructing the Priority Queue.

new Comparator<Edge>({
    @Override
    public int compare(Edge o1, Edge o2) {
        if (o1.val < o2.val) {
            return -1;
        } else if (o2.val < o1.val) {
            return 1;
        } else {
            if (!o1.isLeft && o2.isLeft) {
                return -1;
            } else if (o1.isLeft && !o2.isLeft) {
                return 1;
            } else {
                return 0;
            }
        }
    }
});
