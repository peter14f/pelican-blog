581-shortest_unsorted_continuous_subarray

Mar 18 2018 Tags Two Pointers

LeetCode Problem Link

First the trivial O(nlogn) time and O(n) space solution. You compare the sorted version with the original array.

public int findUnsortedSubarray(int[] nums) {
    if (nums.length <= 1) return 0;
    int[] toSort = new int[nums.length];

    System.arraycopy(nums, 0, toSort, 0, nums.length);
    Arrays.sort(toSort);

    int end = nums.length - 1;
    int start = 0;

    while (end > 0 && toSort[end] == nums[end]) {
        end--;
    }

    while (start < nums.length && toSort[start] == nums[start]) {
        start++;
    }

    if (start > end) {
        return 0;
    } else {
        return end - start + 1;
    }
}

Now this is the O(n) time and O(1) space solution. Once you have the range defined by start and end, figure out the max and min value within that range. Elements that come before start should not be bigger than min and elements that come after end should not be smaller than max.

public int findUnsortedSubarray(int[] nums) {
    if (nums.length <= 1) return 0;

    int end = nums.length - 1;
    int start = 0;

    while (end - 1 >= 0 && nums[end] >= nums[end-1]) {
        end--;
    }

    while (start + 1 < nums.length && nums[start+1] >= nums[start]) {
        start++;
    }

    if (end <= start) {
        return 0;
    }

    int max = nums[start];
    int min = nums[start];

    for (int i=start+1; i<=end; i++) {
        max = Math.max(nums[i], max);
        min = Math.min(nums[i], min);
    }

    for (int i=start-1; i >= 0; i--) {
        if (nums[i] > min) {
            start = i;
        }
    }

    for (int i=end+1; i < nums.length; i++) {
        if (nums[i] < max) {
            end = i;
        }
    }

    return end - start + 1;
}
