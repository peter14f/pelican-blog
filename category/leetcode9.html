268-missing_number

Feb 19 2016 Tags Bucket Sort / Missing Number

LeetCode Problem Link

This problem is trivial if you can use a hashtable. But we're asked to do this in O(n) time and O(1) space.

We can find the sum of integers from 0 to n.

sum = (n+1)*(n+0)/2

The actual sum over the array must be smaller or the same.

The difference is the missing number.

public int missingNumber(int[] nums) {
    int n = nums.length;

    int expectedSum = (n+1)*n/2;

    int actualSum = 0;

    for (int i=0; i<nums.length; i++) {
        actualSum += nums[i];
    }

    int difference = expectedSum - actualSum;

    return difference;
}

263-ugly_number

Feb 19 2016 Tags Prime Factorization / Ugly Number

LeetCode Problem Link

We want to know if prime factors other than 2, 3, and 5 exist for num.

We can use while loops to keep dividing num by 2, 3, and 5 if it's divisible.

public boolean isUgly(int num) {
    if (num < 1)
        return false;

    while ((num % 2) == 0){
        num = num / 2;
    }

    while ((num % 3) == 0){
        num = num / 3;
    }

    while ((num % 5) == 0){
        num = num / 5;
    }

    return (num == 1);
}
