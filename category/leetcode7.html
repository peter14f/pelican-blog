280-wiggle_sort

Feb 21 2016 Tags Sorting

LeetCode Problem Link

The straightforward solution takes O(n 2) time.

Just continue replacing the current element with either the largest or the smallest element in the remaining array.

public void wiggleSort(int[] nums) {

    for (int i=0; i<nums.length; i++) {
        if (i%2 == 0) {
            // find small
            replaceLWithSmallest(nums, i);
        }
        else {
            // find big
            replaceLWithBiggest(nums, i);
        }
    }
}

private void replaceLWithSmallest(int[] nums, int l) {
    int min = nums[l];
    int minIndex = l;

    for (int i=l; i<nums.length; i++) {
        if (nums[i] < min) {
            min = nums[i];
            minIndex = i;
        }
    }

    int t = nums[l];
    nums[l] = min;
    nums[minIndex] = t;
}

278-first_bad_version

Feb 20 2016 Tags Binary Search

LeetCode Problem Link

There are n versions labeled from 1 to n.

O(n) solution would be to try call isBadVersion() for 0, 1, 2, ... .

The O(logn) solution uses binary search.

public int firstBadVersion(int n) {
    int l = 1;
    int h = n;

    int minFound = n;

    while (l <= h) {
        int m = l + (h-l)/2;

        boolean bad = isBadVersion(m);

        if (bad) {
            minFound = Math.min(minFound, m);
            // could have a lower bad version
            h = m - 1;
        }
        else {
            // could have a higher bad version
            l = m + 1;
        }
    }

    return minFound;
}

274-h_index

Feb 20 2016 Tags H-Index

LeetCode Problem Link

The input array's size n is the number of papers in total.

The straightforward solution starts checking h-index value h at n and decrement h and continues the check until h becomes zero.

This would take O(n 2) time.

public int hIndex(int[] citations) {
    int n = citations.length;

    int h = n;

    while (h > 0) {

        if (satisfyH(h, citations)) {
            return h;
        }

        h--;
    }

    return h;
}

private boolean satisfyH(int h, int[] citations) {
    int numEnough = 0; // # papers having >= h citations

    for (int i=0; i<citations.length; i++) {
        if (citations[i] >= h)
            numEnough++;
    }

    return numEnough >= h;
}
