094-binary_tree_inorder_traversal

Nov 18 2015 Tags Binary Trees / Binary Tree Traversal / Inorder Traversal

LeetCode Problem Link

I solved this problem using a stack and a while loop.

public List<Integer> inorderTraversal(TreeNode root) {

    List<Integer> ans = new ArrayList<Integer>();

    if (root==null)
        return ans;

    Stack<TreeNode> stk = new Stack<TreeNode>();

    stk.push(root);
    TreeNode cur = root.left;

    while (cur != null || !stk.isEmpty()) {

        if (cur != null) {
            stk.push(cur);
            cur = cur.left;
        }
        else {
            TreeNode top = stk.pop();
            ans.add(top.val);
            cur = top.right;
        }
    }

    return ans;
}

I chose to do this problem iteratively. Reversing a linked list iteratively requires using a prev pointer.

public ListNode reverseBetween(ListNode head, int m, int n) {

    if (head == null || m==n) {
        return head;
    }

    if (m==1) {
        head = reverseBetween(head, n-m);
        return head;
    }
    else if (m > 1) {
        int i=1;
        ListNode cur = head;
        ListNode prev = null;

        do {
            prev = cur;
            cur = cur.next;
            i++;
        } while (i < m);

        prev.next = reverseBetween(cur, n-m);
        return head;
    }
    else
        throw new IllegalArgumentException();

}

088-merge_sorted_array

Nov 15 2015

LeetCode Problem Link

num1 actually has n slots after the first m elements. We must insert from the end, so that we don't lose the elements in num1.

public void merge(int[] nums1, int m, int[] nums2, int n) {
    int p1 = m-1;
    int p2 = n-1;
    int i = m+n-1;

    while (p1 >= 0 && p2 >= 0) {

        if (nums1[p1] >= nums2[p2]) {
            nums1[i] = nums1[p1];
            p1--;
        }
        else {
            nums1[i] = nums2[p2];
            p2--;
        }
        i--;
    }

    // done inserting elements from array2
    if (p1 >= 0)
        return;

    // done inserting elements from array1
    while (i >= 0) {
        nums1[i] = nums2[p2];
        p2--;
        i--;
    }
}
