334-increasing_triplet_subsequence

LeetCode Problem Link

Trivial solution is to use 3 nested for loops. But the time complexity is O(n 3).

public boolean increasingTriplet(int[] nums) {
    for (int i=0; i<nums.length-2; i++) {
        for (int j=i+1; j<nums.length-1; j++) {
            if (nums[j] <= nums[i])
                continue;

            for (int k=j+1; k<nums.length; k++) {
                if (nums[k] <= nums[j])
                    continue;

                if (nums[k] > nums[j] && nums[j] > nums[i])
                    return true;
            }
        }
    }

    return false;
}

Maintain smallestSeen and secondSmallestSeen s you traverse the array.

327-count_of_range_sum

Feb 28 2016 Tags Segment Tree

LeetCode Problem Link

public int countRangeSum(int[] nums, int lower, int upper) {
    int n = nums.length;

    int count = 0;

    for (int i=0; i<n; i++) {
        int curSum = nums[i];

        if (curSum >= lower && curSum <= upper)
            count++;

        for (int j=i+1; j<n; j++) {
            curSum += nums[j];

            if (curSum >= lower && curSum <= upper)
                count++;
        }
    }

    return count;
}

This is the trivial O(n 2) time solution that does not check for overflow
