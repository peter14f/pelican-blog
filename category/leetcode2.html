231-power_of_two

Feb 11 2016 Tags Power

LeetCode Problem Link

Pretty straightforward problem. Just remember that a power of 2 number cannot be negative.

public boolean isPowerOfTwo(int n) {
    // number of bits set
    int numBits = 0;

    do {
        if ((n & 1) == 1) {
            numBits++;
            if (numBits > 1)
                return false;
        }
        n = n >> 1;
    } while (n != 0);

    return (numBits==1);
}

227-invert_binary_tree

LeetCode Problem Link

Use recursion. The base case is the empty true where the tree root is simply returned.

public TreeNode invertTree(TreeNode root) {
    if (root == null)
        return null;

    invertTree(root.left);
    invertTree(root.right);

    TreeNode oldLeft = root.left;
    root.left = root.right;
    root.right = oldLeft;

    return root;
}
