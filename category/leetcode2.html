739-daily_temperatures

Jul 10 2018 Tags Stacks

LeetCode Problem Link

The trivial solution would take O(n2) time. This problem reminds of 84-largest_rectangle.

public int[] dailyTemperatures2(int[] temperatures) {
    Stack<Integer> stk = new Stack<Integer>(); // stores the index
    int[] ans = new int[temperatures.length];
    int i=0;
    while (!stk.isEmpty() || i<temperatures.length) {
        if (stk.isEmpty()) {
            stk.push(i);
            i++;
            continue;
        }

        if (i >= temperatures.length || temperatures[i] > temperatures[stk.peek()]) {
            int index = stk.pop();
            if (i >= temperatures.length) {
                ans[index] = 0;
            } else {
                ans[index] = i-index;
            }
        } else {
            stk.push(i);
            i++;
        }
    }
    return ans;
}

334-increasing_triplet_subsequence

Feb 28 2016

LeetCode Problem Link

Trivial solution is to use 3 nested for loops. But the time complexity is O(n 3).

public boolean increasingTriplet(int[] nums) {
    for (int i=0; i<nums.length-2; i++) {
        for (int j=i+1; j<nums.length-1; j++) {
            if (nums[j] <= nums[i])
                continue;

            for (int k=j+1; k<nums.length; k++) {
                if (nums[k] <= nums[j])
                    continue;

                if (nums[k] > nums[j] && nums[j] > nums[i])
                    return true;
            }
        }
    }

    return false;
}

Maintain smallestSeen and secondSmallestSeen s you traverse the array.
