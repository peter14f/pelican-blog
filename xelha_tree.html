xelha_tree

Feb 28 2016 Tags Recursion / Inorder Traversal

A Xelha tree is a binary tree. Every node is smaller than (or equal to) its children.

Write a method that takes an array of integers that returns a xelha tree whose inorder traversal is the same as the input array.

[9, 3, 7, 15, 1, 8, 12]

               1
             /   \
            3     8
           / \     \
          9   7    12
               \
               15

Notice that the root is the smaller element. And its right subtree consists of
elements to its right [8, 12] while its left subtree consists of elements to
its left [9, 3, 7, 15].

static class Node {
    int val;
    Node left;
    Node right;

    public Node(int x) {
        this.val = x;
    }
}

public Node buildTree(int[] nums) {
    return buildTree(nums, 0, nums.length - 1);
}

private Node buildTree(int[] nums, int l, int r) {
    if (l==r) {
        return new Node(nums[l]);
    }
    if (r < l) {
        return null;
    }

    int minIndex = l;
    int minElem = nums[l];

    for (int i=l+1; i<=r; i++) {
        if (nums[i] < minElem) {
            minIndex = i;
            minElem = nums[i];
        }
    }

    Node node = new Node(nums[minIndex]);
    Node left = buildTree(nums, l, minIndex-1);
    Node right = buildTree(nums, minIndex+1, r);

    node.left = left;
    node.right = right;
    return node;
}

This approach takes O(n^2) time in the worst case. We could do better and that involves considering what to do when we already have a tree built want to simply append a new int at the end of the input array. It takes O(n) time.

public Node buildTreee(int[] nums) {
    List<Integer> numList = new ArrayList<>();
    for (int i=0; i<nums.length; i++) {
        numList.add(nums[i]);
    }
    return buildTree(numList, Integer.MIN_VALUE);
}

private Node buildTree(List<Integer> nums, int min) {
    Node root = null;
    while (!nums.isEmpty()) {
        if (nums.get(0) < min) {
            // completed my portion of the tree already
            return root;
        } else if (root == null || nums.get(0) < root.val) {
            Node newNode = new Node(nums.remove(0));
            newNode.left = root;
            root = newNode;
        } else if (nums.get(0) > root.val) {
            root.right = buildTree(nums, root.val);
        }
    }
    return root;
}
