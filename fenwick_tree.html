Feb 28 2016 Tags BIT / Fenwick Tree

Fenwick tree can be easily implemented using an array and is one-based.

16 -10000 [ 1,15]          |
15 - 1111 [15,15] |        |
14 - 1110 [13,14]   |      |
13 - 1101 [13,13] | |      |
12 - 1100 [ 9,12]     |    |
11 - 1011 [11,11] |   |    |
10 - 1010 [ 9,10]   | |    |

 8 - 1000 [ 1, 8]       |  |

 4 - 0100 [ 1, 4]     | |  |
 3 - 0011 [ 3, 3] |   | |  |

 1 - 0001 [ 1, 1] | | | |  |

The value of the LSB of the index number tells us how many elements is being covered.

prefixSum(5) = sum(0, 5) = fenwickSum(6) = tree[6] + tree[4]
