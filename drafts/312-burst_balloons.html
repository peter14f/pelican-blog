312-burst_balloons

Feb 25 2016

LeetCode Problem Link

pad a 1 in the front and back of nums.

int[] numbers = new int[n+2];

Now dp[left][right] is defined as the maximum coins when we burst all balloons between left and right (but not including balloons at left and right).

For example, if the input array is [3, 1, 5, 8], then numbers = [1, 3, 1, 5, 8, 1] and we want to return dp[0][5].
